TamilRockers ProxyTool is to provide direct access to Tamilrockers with the help of ProxySevers.This tool is developed in PythonScript and Embedded with Mozilla FireFox.

Installation Guide:
1.Install Python on your Machine. Click here to Download python
2.Download and Extract TamilRockers ProxyTool.
3.For Windows: Run Windows Installer and then Run Connect Tamilrockers.
3.For Linux:Run python3 Installer.py on Terminal and then Run python3 TaamilRockers.pyon Terminal.
3.For Android: Download and Run APK File and wait until connect to Tamilrockers.

Make sure Internet speed is above 150Kbps.
